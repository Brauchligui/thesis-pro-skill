# Plan de Thèse - Étape 4: Éléments Complémentaires

## VOTRE TÂCHE:

Définir les éléments liminaires et annexes.

## SÉQUENCE:

### 1. Pages Liminaires

```markdown
## Pages Liminaires (à préparer)

### Obligatoires
- [ ] **Page de titre** — Selon format {{institution}}
- [ ] **Résumé en français** — 300-500 mots + mots-clés
- [ ] **Abstract en anglais** — 300-500 mots + keywords
- [ ] **Table des matières** — Générée automatiquement
- [ ] **Liste des figures** — Si applicable
- [ ] **Liste des tableaux** — Si applicable

### Recommandées
- [ ] **Remerciements** — 1 page max
- [ ] **Liste des abréviations** — Si nombreuses
- [ ] **Glossaire** — Si termes techniques

### Notes
- Vérifier le format exact requis par {{institution}}
- Le résumé est crucial: souvent la seule partie lue
```

### 2. Annexes Potentielles

```markdown
## Annexes à Prévoir

### Annexes Méthodologiques
- [ ] Guide d'entretien complet
- [ ] Questionnaire intégral
- [ ] Protocole d'observation
- [ ] Grille de codage

### Annexes de Données
- [ ] Retranscriptions (extraits significatifs)
- [ ] Tableaux de données brutes
- [ ] Outputs statistiques
- [ ] Verbatims complets

### Annexes Documentaires
- [ ] Documents analysés
- [ ] Organigrammes, schémas
- [ ] Chronologies

### Notes sur les Annexes
- Ne mettre en annexe que ce qui est référencé dans le texte
- Les annexes ne comptent pas dans la pagination principale
- Numéroter: Annexe A, Annexe B, etc.
```

### 3. Bibliographie

```markdown
## Bibliographie

### Format
**Style:** {{citation_style}}

### Organisation
Option 1: Ordre alphabétique (le plus courant)
Option 2: Par type de source
Option 3: Par thème

### Rappel Format {{citation_style}}
[Rappel des règles clés du style choisi]

### Outils Recommandés
- Zotero (gratuit)
- Mendeley (gratuit)
- EndNote (payant)
```
