# Plan de Thèse - Étape 1: Choix de la Structure

## VOTRE TÂCHE:

Aider à choisir la structure globale adaptée à la thèse.

## SÉQUENCE:

### 1. Présenter les Structures Types

```markdown
## Structures Classiques de Thèse Professionnelle

### Option A: Structure IMRaD (Sciences, Quanti)
```
Introduction
Méthodes
Résultats
Discussion
```
**Idéal pour:** Recherches quantitatives, tests d'hypothèses

### Option B: Structure en Entonnoir (Business, SHS)
```
Introduction
Partie 1: Revue de Littérature / Cadre Théorique
Partie 2: Méthodologie et Terrain
Partie 3: Résultats et Analyse
Partie 4: Discussion et Recommandations
Conclusion
```
**Idéal pour:** MBA, recherches qualitatives, études de cas

### Option C: Structure Thématique
```
Introduction
Partie 1: [Thème A]
Partie 2: [Thème B]
Partie 3: [Synthèse et Articulation]
Conclusion
```
**Idéal pour:** Sujets multi-dimensionnels

### Option D: Structure Problématisée
```
Introduction
Partie 1: Pourquoi ce problème existe
Partie 2: Comment il se manifeste (terrain)
Partie 3: Comment le résoudre (recommandations)
Conclusion
```
**Idéal pour:** Thèses orientées consulting/action
```

### 2. Recommander une Structure

"Basé sur votre problématique et votre approche méthodologique, **je vous recommande:**

**Structure [X]** parce que:
- [Raison 1]
- [Raison 2]

Qu'en pensez-vous ?"

### 3. Documenter le Choix

```markdown
# Plan de Thèse - {{user_name}}

## Structure Choisie
**Type:** [Structure choisie]
**Justification:** [Raison du choix]

## Grandes Parties
1. Introduction
2. [Partie 1]
3. [Partie 2]
4. [Partie 3]
5. Conclusion
```
