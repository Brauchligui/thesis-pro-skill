## VOTRE TÂCHE:

Définir la stratégie d'échantillonnage appropriée.

## SÉQUENCE:

### 1. Présenter les Stratégies

#### Échantillonnage Qualitatif:

```markdown
## Stratégies d'Échantillonnage Qualitatif

| Stratégie | Description | Quand l'utiliser |
|-----------|-------------|------------------|
| **Purposif** | Sélection raisonnée selon critères | Profils spécifiques recherchés |
| **Boule de neige** | Un participant en recommande un autre | Population difficile d'accès |
| **Théorique** | Jusqu'à saturation | Grounded theory |
| **Cas typiques** | Représentatifs du phénomène | Illustrer le commun |
| **Cas extrêmes** | Révélateurs | Comprendre les limites |
| **Variation maximale** | Diversité des profils | Capturer toutes les perspectives |

**Taille recommandée:**
- Entretiens: 10-30 (saturation)
- Focus groups: 3-5 groupes de 6-10
- Étude de cas: 1-5 cas
```

#### Échantillonnage Quantitatif:

```markdown
## Stratégies d'Échantillonnage Quantitatif

| Stratégie | Description | Avantage |
|-----------|-------------|----------|
| **Aléatoire simple** | Tirage au sort | Représentativité |
| **Stratifié** | Quotas par sous-groupes | Représenter les strates |
| **Convenance** | Accessibilité | Faisabilité |
| **Quotas** | Proportions prédéfinies | Contrôle composition |

**Calcul de taille d'échantillon:**
- Niveau de confiance: 95% → Z = 1.96
- Marge d'erreur: 5%
- Formule: n = (Z² × p × q) / e²
- Exemple: n = (1.96² × 0.5 × 0.5) / 0.05² ≈ 385
```
