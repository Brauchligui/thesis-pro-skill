# Méthodologie - Étape 2: Méthode de Collecte des Données

## RÈGLES D'EXÉCUTION OBLIGATOIRES:

- ✅ Proposer des méthodes adaptées à l'approche choisie
- 📋 Détailler chaque méthode avec avantages/inconvénients
- 🔍 Rechercher des exemples dans le domaine si utile
- 📝 Aider à concevoir les instruments
- ✅ TOUJOURS communiquer dans `{communication_language}`

## VOTRE TÂCHE:

Définir la méthode de collecte de données et aider à concevoir les instruments.

## SÉQUENCE:

### 1. Présenter les Méthodes selon l'Approche

#### Si Approche QUALITATIVE:

```markdown
## Méthodes de Collecte Qualitative

| Méthode | Description | Quand l'utiliser | N typique |
|---------|-------------|------------------|-----------|
| **Entretiens semi-directifs** | Conversation guidée | Explorer en profondeur | 10-30 |
| **Entretiens directifs** | Questions fermées | Comparer des réponses | 20-50 |
| **Focus groups** | Discussion de groupe | Dynamiques collectives | 3-5 groupes |
| **Observation** | Immersion terrain | Comprendre les pratiques | Variable |
| **Étude de cas** | Analyse approfondie | Phénomène complexe | 1-5 cas |
| **Analyse documentaire** | Étude de documents | Données existantes | Variable |

**Pour votre recherche sur {{research_question}}, je recommande:**
[Méthode(s)] parce que [justification]
```

#### Si Approche QUANTITATIVE:

```markdown
## Méthodes de Collecte Quantitative

| Méthode | Description | Quand l'utiliser | N minimum |
|---------|-------------|------------------|-----------|
| **Questionnaire en ligne** | Auto-administré | Large diffusion | 100-500+ |
| **Questionnaire administré** | Face-à-face/téléphone | Contrôle échantillon | 50-200 |
| **Expérimentation** | Manipulation variables | Tester causalité | Variable |
| **Données secondaires** | Bases existantes | Variables existantes | Variable |

**Pour votre recherche, je recommande:**
[Méthode] parce que [justification]
```

### 2. Concevoir l'Instrument

#### Si Entretiens:

```markdown
## Guide d'Entretien - Structure Type

### Introduction (5 min)
- Présentation de la recherche
- Rappel confidentialité
- Demande d'accord enregistrement

### Questions d'ouverture (5-10 min)
- Parcours, contexte
- Mise en confiance

### Questions sur le thème central (20-30 min)
1. [Question principale 1]
   - Relance: [...]
2. [Question principale 2]
   - Relance: [...]
3. [Question principale 3]
   - Relance: [...]

### Questions d'approfondissement (10-15 min)
- Exemples concrets
- Situations vécues

### Synthèse (5 min)
- Résumé des points clés
- Dernières réflexions
- Remerciements
```

#### Si Questionnaire:

```markdown
## Structure du Questionnaire

### Section 1: Introduction
- Objectif de l'étude
- Temps estimé
- Confidentialité

### Section 2: Questions filtre
- Critères d'éligibilité

### Section 3: Variables principales
**Échelles validées à utiliser:**
- [Échelle 1] — [Auteur, Année]
- [Échelle 2] — [Auteur, Année]

### Section 4: Variables de contrôle
- [Variable 1]
- [Variable 2]

### Section 5: Données sociodémographiques
- Âge, genre, fonction, ancienneté, etc.

### Recommandations:
- Pré-test sur 10-20 personnes
- Temps de réponse < 15 minutes
```
