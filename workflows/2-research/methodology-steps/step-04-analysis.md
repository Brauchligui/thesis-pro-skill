## SÉQUENCE:

### 1. Présenter les Méthodes d'Analyse

#### Analyse Qualitative:

```markdown
## Méthodes d'Analyse Qualitative

| Méthode | Description | Quand l'utiliser |
|---------|-------------|------------------|
| **Analyse thématique** | Identifier des thèmes récurrents | La plus courante, flexible |
| **Analyse de contenu** | Catégorisation systématique | Données textuelles structurées |
| **Grounded Theory** | Théorie émergente des données | Développer une nouvelle théorie |
| **IPA** | Interprétation phénoménologique | Comprendre l'expérience vécue |
| **Analyse narrative** | Structure des récits | Histoires de vie |

### Étapes de l'Analyse Thématique (Braun & Clarke)
1. Familiarisation avec les données
2. Génération des codes initiaux
3. Recherche de thèmes
4. Révision des thèmes
5. Définition et nomination des thèmes
6. Production du rapport

### Logiciels recommandés
- NVivo (payant, complet)
- ATLAS.ti (payant)
- MAXQDA (payant)
- Excel/Word (gratuit, basique)
```

#### Analyse Quantitative:

```markdown
## Méthodes d'Analyse Quantitative

### Analyses Descriptives
- Moyennes, écarts-types
- Fréquences, pourcentages
- Tableaux croisés

### Analyses Inférentielles

| Test | Quand l'utiliser |
|------|------------------|
| **Test t** | Comparer 2 moyennes |
| **ANOVA** | Comparer 3+ moyennes |
| **Chi-2** | Relations entre variables catégorielles |
| **Corrélation** | Force de la relation entre 2 variables |
| **Régression** | Prédire une variable par d'autres |

### Analyses Avancées
- Analyse factorielle (ACP, AFC)
- Modèles d'équations structurelles (SEM)
- Analyse de clusters

### Logiciels recommandés
- SPSS (payant, intuitif)
- R (gratuit, puissant)
- JASP (gratuit, moderne)
- Excel (basique)
```
