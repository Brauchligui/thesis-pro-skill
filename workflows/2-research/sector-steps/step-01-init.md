# Analyse Sectorielle - Étape 1: Cadrage et Sources

## VOTRE TÂCHE:

Définir le périmètre et identifier les sources de données sectorielles.

## SÉQUENCE:

### 1. Confirmer le Périmètre

"**Analyse sectorielle: {{sector}}**

**Périmètre:**
- Secteur: {{sector}}
- Géographie: {{geography}}
- Période: Données récentes (3-5 ans) + tendances

**Objectif:** {{analysis_goal}}"

### 2. Identifier les Sources

```markdown
## Sources de Données Sectorielles

### Données Macro-économiques
| Source | Type | Accès |
|--------|------|-------|
| INSEE | Statistiques France | Gratuit |
| Eurostat | Statistiques Europe | Gratuit |
| Banque Mondiale | Monde | Gratuit |
| OCDE | International | Gratuit |

### Études Sectorielles
| Source | Type | Accès |
|--------|------|-------|
| Xerfi | Études sectorielles FR | Payant |
| Statista | Statistiques | Partiellement gratuit |
| IBISWorld | Industries | Payant |

### Rapports Professionnels
- Big 4 (Deloitte, PwC, EY, KPMG)
- McKinsey, BCG, Bain
- Fédérations professionnelles du secteur

### Sources Académiques
- Articles de recherche sur le secteur
- Thèses sur le même secteur
```

### 3. Préparer les Requêtes

```markdown
## Requêtes de Recherche

### Taille et Croissance
- "{{sector}} market size {{geography}} 2024"
- "{{sector}} industry growth rate"
- "taille marché {{sector}} France"

### Tendances
- "{{sector}} trends 2024 2025"
- "{{sector}} tendances évolution"

### Acteurs
- "{{sector}} major players companies"
- "{{sector}} leaders market share"

### Réglementation
- "{{sector}} regulation {{geography}}"
- "réglementation {{sector}} France"
```
