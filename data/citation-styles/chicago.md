# Chicago Citation Style (17th Edition)

## Description
Deux systèmes : Notes-Bibliographie (humanities) et Auteur-Date (sciences). En école de commerce, on utilise souvent Auteur-Date.

---

## Système Auteur-Date (recommandé pour business)

### Dans le texte
- (Kahneman 2011)
- (Kahneman 2011, 45) pour une page spécifique
- Kahneman (2011) affirme que...

### Livre
```
Nom, Prénom. Année. *Titre en italique*. Lieu: Éditeur.
```
**Exemple :**
Kahneman, Daniel. 2011. *Thinking, Fast and Slow*. New York: Farrar, Straus and Giroux.

### Article de journal
```
Nom, Prénom, et Prénom Nom. Année. "Titre de l'article." *Titre du Journal* Volume (Numéro): Pages. DOI.
```
**Exemple :**
Tversky, Amos, and Daniel Kahneman. 1974. "Judgment under Uncertainty: Heuristics and Biases." *Science* 185 (4157): 1124–31.

---

## Système Notes-Bibliographie

### Note de bas de page (première citation)
```
Prénom Nom, *Titre* (Lieu: Éditeur, Année), page.
```

### Note abrégée (citations suivantes)
```
Nom, *Titre abrégé*, page.
```

### Bibliographie
```
Nom, Prénom. *Titre*. Lieu: Éditeur, Année.
```

## Règles importantes
1. Virgule avant l'année en Auteur-Date
2. Guillemets pour les articles, italique pour les livres
3. Ordre alphabétique dans la bibliographie
4. En-têtes "Bibliography" ou "References"
