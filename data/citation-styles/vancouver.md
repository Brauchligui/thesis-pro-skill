# Vancouver Citation Style

## Description
Style numérique séquentiel, utilisé principalement dans les domaines médicaux et biomédicaux. Les références sont numérotées dans l'ordre de leur première apparition dans le texte.

## Format de base

### Dans le texte
- Numéro entre parenthèses ou en exposant : (1) ou ¹
- Plusieurs sources : (1,2) ou (1-3) pour une séquence
- Exemple : "Des études récentes ont démontré cette corrélation (1,2)."

### Livre
```
Auteur AA, Auteur BB. Titre du livre. Édition. Lieu: Éditeur; Année.
```
**Exemple :**
Kahneman D. Thinking, Fast and Slow. New York: Farrar, Straus and Giroux; 2011.

### Article de journal
```
Auteur AA, Auteur BB. Titre de l'article. Nom du Journal Abrégé. Année;Volume(Numéro):Pages.
```
**Exemple :**
Tversky A, Kahneman D. Judgment under Uncertainty: Heuristics and Biases. Science. 1974;185(4157):1124-1131.

### Article en ligne avec DOI
```
Auteur AA. Titre de l'article. Nom du Journal. Année;Volume(Numéro):Pages. doi:xxxxx
```

### Site web
```
Auteur/Organisation. Titre de la page [Internet]. Lieu: Éditeur; Date [cité le Date d'accès]. Disponible sur: URL
```

### Thèse/Mémoire
```
Auteur AA. Titre de la thèse [thèse]. Lieu: Université; Année.
```

## Règles importantes
1. Maximum 6 auteurs, puis "et al."
2. Abréger les noms de journaux selon Index Medicus
3. Pas d'italique pour les titres
4. Numérotation continue tout au long du document
