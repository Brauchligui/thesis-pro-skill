# Harvard Citation Style

## Description
Style auteur-date largement utilisé dans les universités britanniques et européennes, notamment en business et sciences sociales. Pas de standard unique - vérifier les exigences de votre institution.

## Format de base

### Dans le texte
- Un auteur : (Kahneman, 2011) ou Kahneman (2011)
- Deux auteurs : (Tversky and Kahneman, 1974)
- Trois auteurs ou plus : (Tversky et al., 1974)
- Citation directe : (Kahneman, 2011, p. 45)
- Plusieurs sources : (Kahneman, 2011; Ariely, 2008)

### Livre
```
Nom, Initiale(s). (Année) *Titre du livre en italique*. Édition. Lieu: Éditeur.
```
**Exemple :**
Kahneman, D. (2011) *Thinking, fast and slow*. New York: Farrar, Straus and Giroux.

### Chapitre de livre édité
```
Nom, Initiale(s). (Année) 'Titre du chapitre', in Initiale(s). Nom (ed.) *Titre du livre*. Lieu: Éditeur, pp. x-x.
```

### Article de journal
```
Nom, Initiale(s). (Année) 'Titre de l'article', *Titre du Journal*, Volume(Numéro), pp. x-x.
```
**Exemple :**
Tversky, A. and Kahneman, D. (1974) 'Judgment under uncertainty: Heuristics and biases', *Science*, 185(4157), pp. 1124-1131.

### Article en ligne
```
Nom, Initiale(s). (Année) 'Titre', *Journal*, Volume(Numéro), pp. x-x. Available at: URL (Accessed: Date).
```

### Site web
```
Auteur/Organisation (Année) *Titre de la page*. Available at: URL (Accessed: Date).
```

### Rapport d'entreprise
```
Nom de l'organisation (Année) *Titre du rapport*. Lieu: Éditeur.
```

## Règles importantes
1. Ordre alphabétique par nom d'auteur
2. Retrait suspendu pour chaque entrée
3. "and" entre deux auteurs (pas &)
4. Guillemets simples pour les titres d'articles
5. Toujours inclure la date d'accès pour les sources en ligne
