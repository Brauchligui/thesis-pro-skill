# APA Citation Style (7th Edition)

## Description
Style auteur-date développé par l'American Psychological Association. Très utilisé en sciences sociales, psychologie, éducation et business.

## Format de base

### Dans le texte
- Un auteur : (Kahneman, 2011) ou Kahneman (2011)
- Deux auteurs : (Tversky & Kahneman, 1974)
- Trois auteurs ou plus : (Tversky et al., 1974)
- Citation directe : (Kahneman, 2011, p. 45)

### Livre
```
Auteur, A. A. (Année). *Titre du livre en italique* (Édition). Éditeur.
```
**Exemple :**
Kahneman, D. (2011). *Thinking, fast and slow*. Farrar, Straus and Giroux.

### Chapitre de livre
```
Auteur, A. A. (Année). Titre du chapitre. Dans A. Éditeur (Éd.), *Titre du livre* (p. xx-xx). Éditeur.
```

### Article de journal
```
Auteur, A. A., & Auteur, B. B. (Année). Titre de l'article. *Titre du Journal*, *Volume*(Numéro), pages. https://doi.org/xxxxx
```
**Exemple :**
Tversky, A., & Kahneman, D. (1974). Judgment under uncertainty: Heuristics and biases. *Science*, *185*(4157), 1124-1131.

### Site web
```
Auteur, A. A. (Date). *Titre de la page*. Nom du site. URL
```

### Rapport
```
Organisation. (Année). *Titre du rapport* (Numéro de rapport si applicable). URL
```

## Règles importantes
1. Double interligne dans la bibliographie
2. Retrait suspendu (hanging indent) de 1,27 cm
3. Maximum 20 auteurs, puis "..." et le dernier auteur
4. Ordre alphabétique par nom de famille
5. Utiliser "&" dans les parenthèses, "et" dans le texte courant
